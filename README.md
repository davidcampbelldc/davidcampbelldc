# David Campbell

**Performance Engineering x AI | 25+ Years | Founder of [LoadMagic.ai](https://loadmagic.ai)**

Building AI-powered tools that eliminate the grunt work from performance testing. Currently shipping production agentic AI systems with Claude Code and the Anthropic SDK.

## What I'm Building

- **[LoadMagic.ai](https://loadmagic.ai)** -- AI-powered JMeter plugin that automates test script creation, correlation, and QA. Turns hours of manual work into minutes.
- **Production agentic AI systems** -- multi-channel AI assistants with tool use, confirmation flows, and autonomous background processing.
- **Hybrid RAG pipelines** -- FTS5 + vector search over domain-specific knowledge bases.
